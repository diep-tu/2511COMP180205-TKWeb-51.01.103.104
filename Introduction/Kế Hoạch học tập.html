Tuần 1: Làm quen và xây dựng nền tảng kiến thức cơ bản

  1. Thiết kế web
    • Giới thiệu về HTML, CSS cơ bản.
    • Thực hành tạo trang web tĩnh đơn giản.
  2. Lập trình cơ bản
    • Khái niệm lập trình, biến, kiểu dữ liệu.

    • Tập hợp, ánh xạ, quan hệ.
    • Các phép toán trên tập hợp.

    • Logic mệnh đề, phép toán logic.
    • Bài tập vận dụng logic.

Tuần 2: Phát triển kỹ năng và kiến thức chuyên sâu hơn

  1. Thiết kế web
    • CSS nâng cao, bố cục trang web.
    • Thực hành tạo trang web có bố cục cơ bản.
  2. Lập trình cơ bản
    • Câu lệnh điều kiện, vòng lặp.
    • Viết chương trình xử lý dữ liệu đơn giản.
  3. Cơ sở toán trong tin học
    • Quan hệ, quan hệ tương đương, quan hệ thứ tự.
    • Bài tập minh họa.
  4. Toán rời rạc và ứng dụng
    • Đại số Boole.
    • Bài tập ứng dụng đại số Boole.

Tuần 3: Thực hành và áp dụng kiến thức

  1. Thiết kế web
    • JavaScript cơ bản.
    • Thực hành tạo trang web có tương tác đơn giản.
  2. Lập trình cơ bản
    • Hàm, thủ tục.
    • Viết chương trình có cấu trúc.
  3. Cơ sở toán trong tin học
    • Đồ thị, cây.
    • Bài tập về đồ thị đơn giản.
  4. Toán rời rạc và ứng dụng
    • Đếm và tổ hợp.
    • Bài tập đếm cơ bản.

Tuần 4: Ôn tập, tổng kết và chuẩn bị kiểm tra giữa kì

  1. Tất cả các môn
    • Ôn tập kiến thức đã học.
    • Làm bài tập tổng hợp, đề thi thử.
    • Tham gia thảo luận nhóm, hỏi đáp với giảng viên.
  2. Hoạt động bổ trợ
    • Lập kế hoạch học tập cá nhân cho học kỳ.
    • Tự đánh giá điểm mạnh, điểm yếu để điều chỉnh phương pháp học.
    • Tham gia các câu lạc bộ học thuật hoặc nhóm học tập.
